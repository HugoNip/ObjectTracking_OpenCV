# Object Tracking 

## OpenCV Version

- OpenCV 4.3.0

## Steps to build

To compile on Linux, mac or Windows using Mingw

```
mkdir build
cd build
cmake ..
make
```

## Executables

The following applications are generated.

```
./camshiftTracker  
./coloredObjectTracker  
./farnebackTracker  
./goodFeaturesToTrack  
./harrisCornersTracker  
./lucasKanadeTracker
```
